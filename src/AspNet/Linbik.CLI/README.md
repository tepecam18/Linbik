# Linbik.CLI

Command-line tool for bootstrapping Linbik projects and managing service registrations.

## 📦 Installation

```bash
dotnet tool install --global Linbik.Cli
```

## 🚀 Commands

### `linbik init`

Initialize a new Linbik project. Scaffolds configuration files and registers the service.

```bash
linbik init
```

### `linbik status`

Check the current service registration status and connectivity.

```bash
linbik status
```

### `linbik export-config`

Export the current Linbik configuration.

```bash
linbik export-config
```

## 🔧 Requirements

- .NET 10.0 SDK or later
- A Linbik account at [linbik.com](https://linbik.com)
